# Miniboard Soundboard Player
Miniboard is a simple soundboard player without all the paid shenanigans. Best paired with VB Audio cables.

Everyone gets the right to use a soundboard without a limit or a monthly subscription. No ads, no tracking, no data collection, just a simple press and play.

Miniboard allows you to use custom sounds using the VB Audio cables without the need of voicemeeter.

## Features
- Normal & Layer Mode : Allows you to layer sounds ontop of each other, and normal mode to stop the sound if it detects another sound
- Mp3, Wav, Flac, Ogg support : Use your favorite format with ease of mind.
- Monitor : Monitor your voice and sounds. or if you feel ugly only sounds.
- Open Source : Audit, modify, and use the codebase as you please
- Light-f*cking-fast : Easy on your hardware, minimal footprint and memory and it just works.
- MB Audio Engine : No need for a mixer, no need for drivers, you download, works. Caveman language works.
